利用GitHub在Markdown中插入图片

By Codegitz at: 2017-07-10
git

1.新建一个git仓库

在github中新建一个repo，不要勾选Initalize this repository with a README选项，如建立一个MarkdownPhotos的仓库

2.新建本地储存图片的文件夹

在本地电脑新建文件夹将图片放入文件夹中，如新建ImageGithubToMarkdown文件夹

3.在对应文件夹建立本地仓库

4.Push到github保存


完成后的图片为

5.复制图片地址

在Google浏览器打开GitHub

右键点击图片，选择在“新标签页中打开图片”

复制地址栏图片地址
或者直接右击图片，“复制图片地址”，两种操作得到的图片地址均可

两个地址，指向同一个图片

6.在markdown中引用需要的图片地址

按照markdown语法格式引用图片地址即可显示图片
