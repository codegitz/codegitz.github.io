怎么将GitHub中的项目导入Eclipse

By Codegitz at: 2017-11-13
java

导入项目

首先我们再Eclipse中打开Git视图，如图打开，选择Git，点击OK


然后找到clone a Git Repository,如图红色框所指


点击然后会出现如下界面，URL那一栏需要填写你要克隆的项目的地址，我们随便去GitHub上找个项目

点绿色框clone or download

复制出现的地址

贴到刚刚的URL框中，就会变成这样，下面的user和passwords是我个人的，有所不同，这个没有关系，点next

一般克隆项目我们只要克隆master分支，勾上，next

红色框内为存放地址，自己视情况而定，一般放到Eclipse工作空间里，点finish

然后我们的Git Repository会出现我们刚刚导入的项目

展开，Working Directory选择项目文件，右击，import project


如图，选择第一个，next

勾上，点finish

这时候工程目录就会出现我们刚刚导入的项目了

打开，可以完整的看到项目的结构，以及里面的文件以及jar包

修改配置，去除错误

刚刚导入的项目，一般不会正常的，会出现许多奇奇怪怪的错误，有时候是jar包丢失，有时候是配置XML文件出错，这时候就要根据对应的错误去寻找解决办法了，这里就不一一叙述了。
