Mysql报错

By Codegitz at: 2018-04-13
Mysql

Mysql报错：Unable to connect to remote host. Catalog download has failed.

问题分析
该问题是由于新版本的 Mysql 在安装的时候会自动在 Windows 的“任务计划程序”中安装一个定时任务，任务的触发时间为每天上午十点。

解决方式
找到相应的定时任务，并禁用该任务。
